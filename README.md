## Introduction

This repository contains our [ECMAScript 5](es5/), [ECMAScript 6](es6/), and
[ECMAScript 7](es7/) style guides, as well as our JSHint and ESLint resource
files for each language, which can be installed via NPM and extended for your
own uses.

For more information, as well as installation instructions, see the
README files in the directory of the ES version you wish to use.
